# PDF변환기 요구사항 요약

- Windows 10/11 64비트, 관리자 권한 없이 실행
- EXE 및 소스코드 제공
- HWP/HWPX, Excel, Word, PowerPoint 지원
- 파일 추가, 폴더 추가, 하위 폴더, 드래그 앤 드롭
- 파일별 개별 PDF 생성
- 사용자가 저장 폴더 지정
- 동일 파일명은 `(1)`, `(2)` 자동 증가
- 원본 용지, 방향, 배치, 내용, 수정일 보존
- 파일별 페이지 범위 목록 직접 입력
- 컬러/흑백 및 품질 선택
- PowerPoint 숨김 슬라이드 제외
- 흑백은 문서 설정을 바꾸지 않고 출력 계층에서 처리
- 실패 파일이 있어도 계속 진행하고 결과창 및 TXT 로그 생성
- 사용자가 버튼을 눌렀을 때만 현재 Windows의 설치 프린터 목록 조회
- 프린터 이름을 고정하지 않고 PC마다 동적으로 표시
- 내장 PDF 엔진 또는 선택한 PDF 가상 프린터 출력 방식 선택
- 선택한 프린터 드라이버의 인쇄 기본 설정 창 열기
- PDF 오류 검사 항목을 `NG/N.G`, `##`, `??`, 사용자 지정 문구로 개별 선택
- 오류가 발견된 PDF 페이지, 오류 문구 및 발견 횟수를 결과 화면에 표시
- 텍스트 검색 불가 페이지를 결과 화면에 별도 표시
- 파일 목록의 파일명, 페이지 범위, 원본 경로 직접 수정
- 우클릭으로 선택 셀, 파일명, 원본 경로 및 행 전체 복사
- 파일별 오류 검사 체크박스로 검사 실행 또는 미실행 선택
- 엑셀 주요 오류 `#####`, `#N/A`, `#DIV/0!`, `#NAME?`, `#VALUE!` 일괄 검사 선택
- 체크된 파일에만 사용자가 선택한 오류 항목 검사
- 변환 결과 표에서 PDF 오류 검사 내용을 우클릭으로 복사
