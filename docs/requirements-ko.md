# PDF변환기 요구사항 요약

- Windows 10/11 64비트, 관리자 권한 없이 실행
- EXE 및 소스코드 제공
- HWP/HWPX, Excel, Word, PowerPoint 지원
- 파일 추가, 폴더 추가, 하위 폴더, 드래그 앤 드롭
- 파일별 개별 PDF 생성
- 사용자가 저장 폴더 지정
- 동일 파일명은 `(1)`, `(2)` 자동 증가
- 원본 용지, 방향, 배치, 내용, 수정일 보존
- 파일별 페이지 범위 목록 직접 입력
- 컬러/흑백 및 품질 선택
- PowerPoint 숨김 슬라이드 제외
- 흑백은 문서 설정을 바꾸지 않고 출력 계층에서 처리
- 실패 파일이 있어도 계속 진행하고 결과창 및 TXT 로그 생성
