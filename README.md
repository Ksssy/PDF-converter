# PDF변환기

Windows 10/11에서 HWP/HWPX, Excel, Word, PowerPoint 문서를 개별 PDF로 일괄 변환하기 위한 데스크톱 프로그램입니다.

## 현재 구현 범위 (v0.7.0)

- PySide6 기반 GUI
- 파일 여러 개 추가
- 폴더 및 하위 폴더 검색
- 드래그 앤 드롭
- 파일 삭제, 전체 삭제, 순서 위/아래 이동
- 파일별 페이지 범위 직접 입력
- 저장 폴더 선택
- 품질 및 컬러/흑백 선택 UI
- 전체 진행률 및 파일별 상태 표시 기반
- 설정 JSON 저장
- TXT 로그 기반
- 확장자별 변환기 플러그인 구조
- Word, Excel, PowerPoint COM PDF 변환
- HWP/HWPX 한글 자동화 PDF 변환
- 여러 파일 변환 중 실패한 파일을 기록하고 다음 파일 계속 처리
- 동일한 PDF 파일명은 `(1)`, `(2)` 자동 증가
- 페이지 범위(`1,3-5`) 적용
- PowerPoint 숨김 슬라이드 제외
- 일시정지 및 중지
- 모든 지원 문서의 컬러/흑백 PDF 출력
- 흑백 변환 시 검색 가능한 텍스트와 벡터 그래픽 유지
- 최소용량 선택 시 PDF 이미지 압축
- 파일별 성공·실패·저장 위치·오류를 보여주는 변환 결과 화면
- Python 설치 없이 실행 가능한 단일 `PDFConverter.exe`
- 버튼을 눌렀을 때만 Windows 설치 프린터를 조회하는 수동 프린터 인식
- 다른 PC의 프린터 목록을 고정값 없이 동적으로 표시
- 선택한 PDF 가상 프린터를 통한 PDF 재출력
- 선택한 출력 방식과 프린터 이름 설정 저장
- 선택한 프린터 드라이버의 `프린터 속성`(인쇄 기본 설정) 열기
- `NG/N.G`, `##`, `??` 및 사용자 지정 문구를 선택하여 PDF 오류 검사
- PDF 오류의 페이지 번호, 문구 및 발견 횟수를 변환 결과 화면에 표시
- 파일 목록에서 파일명·페이지 범위·원본 경로 직접 수정
- 파일 목록 우클릭으로 셀·파일명·원본 경로·행 전체 복사
- 파일별 `오류 검사` 체크박스로 검사 실행·미실행 선택
- `엑셀 주요 오류` 선택 시 `#####`, `#N/A`, `#DIV/0!`, `#NAME?`, `#VALUE!` 검사
- 변환 결과의 PDF 오류 검사 내용을 우클릭으로 복사

한글 자동화 보안 모듈이 설치되지 않은 PC에서는 첫 변환 때 한글의 파일 접근 승인 창이 표시될 수 있습니다.
현재 빌드는 핵심 변환, 흑백 출력, 결과 확인 및 단일 EXE 배포까지 지원합니다.

## PDF 품질 차이

### 내장 PDF 엔진

- **최소용량**: 180DPI를 넘는 이미지를 120DPI, 품질 70 수준으로 압축합니다.
- **일반**: 360DPI를 넘는 이미지를 220DPI, 품질 85 수준으로 정리합니다.
- **고품질**: 이미지를 추가 압축하지 않고 문서 프로그램이 만든 PDF 품질을 유지합니다.

모든 단계에서 텍스트와 벡터 도형은 이미지 압축 대상이 아니며 원래 구조를 유지합니다.

### 선택한 PDF 프린터 사용

- **최소용량**: 200DPI로 프린터에 출력합니다.
- **일반**: 300DPI로 프린터에 출력합니다.
- **고품질**: 450DPI로 프린터에 출력합니다.

`프린터 인식` 버튼을 누르면 그 시점에 Windows에 설치된 프린터를 조회합니다.
프로그램을 시작할 때는 자동으로 프린터를 검색하지 않습니다. PDF 가상 프린터가 아닌 종이 프린터를
선택하면 실제 인쇄될 수 있으므로, 프린터 방식은 PDF 저장 기능이 있는 가상 프린터에서만 사용해야 합니다.
프린터 방식은 드라이버 특성에 따라 생성된 PDF의 텍스트 검색이 지원되지 않을 수 있습니다.

`프린터 속성` 버튼은 선택한 Windows 프린터 드라이버의 인쇄 기본 설정을 엽니다.
드라이버가 지원하는 용지, 방향, PDF 저장 및 출력 관련 설정을 변경한 뒤 변환할 수 있습니다.

## PDF 오류 검사

- `NG / N.G`, `##`, `??`를 각각 선택하거나 해제할 수 있습니다.
- `엑셀 주요 오류`는 `#####`, `#N/A`, `#DIV/0!`, `#NAME?`, `#VALUE!`를 함께 검사합니다.
- `직접 입력`에 `#VALUE!, #DIV/0!, ERROR`처럼 쉼표로 구분한 문구를 추가할 수 있습니다.
- 파일 목록의 `오류 검사`가 체크된 파일에만 선택한 항목을 검사합니다.
- 아무 오류 항목도 선택하지 않으면 파일 체크 여부와 관계없이 검사를 건너뜁니다.
- 변환 결과에서 `2페이지: NG / N.G 3건`처럼 페이지와 발견 횟수를 표시합니다.
- 텍스트가 없는 페이지는 `텍스트 검색 불가`로 별도 표시합니다.
- 선택한 PDF 프린터를 사용할 때는 드라이버가 텍스트를 이미지로 바꾸기 전 내부 PDF를 검사합니다.

파일 목록의 파일명, 페이지 범위, 원본 경로는 직접 입력해 수정할 수 있습니다.
표에서 우클릭하면 선택 셀, 파일명, 원본 경로 또는 행 전체를 클립보드로 복사할 수 있습니다.
변환 결과 표에서도 우클릭하여 선택 셀, PDF 오류 검사 내용 또는 행 전체를 복사할 수 있습니다.

## 개발 환경

- Python 3.11 이상
- Windows 10/11 64비트
- PySide6
- pywin32

## 실행

```powershell
python -m venv .venv
.venv\Scripts\activate
pip install -e .[dev]
pdf-converter
```

## 독립 실행형 EXE 빌드

```powershell
.venv\Scripts\python.exe -m PyInstaller --clean --noconfirm PDFConverter.spec
```

빌드 결과는 `dist\PDFConverter.exe`에 생성됩니다. 이 파일은 Python 설치 없이 실행할 수 있습니다.
문서 변환을 위해서는 대상 PC에 변환할 문서 형식에 맞는 한글 또는 Microsoft Office가 설치되어 있어야 합니다.
